# 🛍️ ShopEasy – Responsive E-Commerce Website

ShopEasy is a modern, responsive e-commerce landing page built using **HTML, CSS, Bootstrap, and JavaScript**. It provides a smooth shopping experience with interactive UI, product listings, cart functionality, and form validations.

## 📌 Features

* 🧾 Responsive design (Mobile + Desktop)
* 🛒 Add to Cart functionality
* 📦 Dynamic product rendering
* 📩 Newsletter subscription form
* 📞 Contact form with validation
* 🎯 Smooth scrolling navigation
* 💡 Interactive UI with animations
* 🧺 Cart modal with checkout simulation

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/shopeasy.git
```

2. Navigate to the project folder:

```bash
cd shopeasy
```

3. Open `index.html` in your browser.

## 🛒 How It Works

* Products are dynamically loaded using JavaScript
* Users can add items to the cart
* Cart updates in real-time
* Checkout is simulated with a simple alert
* Forms include validation for better UX
